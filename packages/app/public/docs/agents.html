Agents 配置

概述

Agents 是可配置的 AI 助手，可以执行特定任务。

Agent 目录结构

.opencode/
├── agent/
│   ├── triage.md      # 问题分类 Agent
│   ├── docs.md        # 文档生成 Agent
│   ├── translator.md  # 翻译 Agent
│   └── duplicate-pr.md # 重复 PR 检测 Agent

Agent 定义

在 .opencode/agent/ 目录中创建 .md 文件来定义 Agent：

---
description: Agent 描述
model: provider/model-name
subtask: true
---
